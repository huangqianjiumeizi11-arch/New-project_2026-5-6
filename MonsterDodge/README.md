# 能天使的大冒险

这是一个使用 Visual Studio、C++ 和 EasyX 制作的俯视角闪避闯关小游戏。第三版已经完成玩家形象、机械竞技场地图风格、Boss 弹幕和升级体系的整体收尾。

## 玩法

- 玩家通过键盘移动角色，鼠标控制近战和子弹方向。
- 地图中有障碍物、追击怪、射击怪、Boss 和随机刷新的回血补给。
- 第 1、2 关清空怪物后进入升级界面。
- 第 3 关击败 Boss 后胜利。

## 当前特色

- 初始化界面标题为“能天使的大冒险”，并使用定制背景图。
- 玩家使用苹果角色形象。
- 地图视觉替换为机械竞技场风格，碰撞逻辑与关卡布局保持一致。
- 近战命中后会加强击退，并提供短暂保护。
- 冲刺改为按下触发，禁止长按持续闪避。
- 冲刺初始持续 14 帧；升级后每次额外增加 4 帧。
- 射击怪固定 25 帧射击一次。
- Boss 固定 25 帧射击一次，每次发射 3 枚弹体：中间一枚瞄准玩家，左右两枚与中间弹保持 30 度夹角。
- 第二关胜利后的“子弹强化”会将玩家射击间隔调整为 20 到 25 帧，子弹伤害保持 1。

## 升级选项

- `1` 攻击力 `+1`
- `2` 延长闪避时间 `+4` 帧
- 第二关后可额外选择：
  - `3` 子弹伤害 `1`，间隔 `20-25` 帧

## 初始菜单

游戏窗口打开后会先进入主菜单，可用鼠标左键选择：

- 开始游戏
- 游戏设置
- 操作说明
- 退出游戏

设置页可控制背景音乐、音量和简单 / 困难模式。

## 操作

- `WASD`：移动
- 鼠标移动：瞄准
- 鼠标左键：近战攻击
- 鼠标右键：发射子弹
- `Space`：触发一次闪避
- `Esc`：游戏中打开暂停菜单
- `R`：失败或胜利后重新开始
- 失败或胜利后按 `Esc`：返回主菜单

## 源码结构

- `MonsterDodge.cpp`：程序入口和主循环
- `Game.h` / `Game.cpp`：游戏状态、关卡、菜单、绘制和更新逻辑
- `MusicManager.h` / `MusicManager.cpp`：背景音乐播放和音量控制
- `Types.h`：基础结构体和枚举
- `third_party/miniaudio/miniaudio.h`：第三方音频库

## 用 Visual Studio 运行

1. 安装 EasyX。
2. 用 Visual Studio 打开 `MonsterDodge.sln`。
3. 选择 `Debug x86` 或 `Release x86`。
4. 按 `F5` 运行。

如果本机 EasyX 或编译库环境与 x86 配置不一致，可再切换平台设置。
